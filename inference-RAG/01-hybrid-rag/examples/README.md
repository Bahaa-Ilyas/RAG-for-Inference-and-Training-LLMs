# Examples

Run the local Hybrid RAG demo:

```bash
python run_example.py
```

Included fixtures:

- `sample_policy.pdf`: generated PDF fixture.
- `scanned_page_ocr.txt`: OCR text fixture for scanned PDF behavior.
- `sample_webpage.html`: webpage fixture with canonical URL semantics.
- `sample_table.csv`: table fixture.
- `tool_response.json`: simulated API/tool response.
